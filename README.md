# Personal Portfolio - Software Engineering Showcase

### Overview
Este proyecto es mi portafolio profesional, diseñado para presentar mi trayectoria y competencias técnicas en el desarrollo de software. El enfoque principal fue crear una experiencia de usuario fluida, minimalista y de alto rendimiento.

### Stack Tecnológico
* **Frontend:** React 19 (Hooks, Functional Components)
* **Lenguaje:** TypeScript (Type-safety estricto)
* **Estilos:** CSS3 con arquitectura de variables dinámicas
* **Despliegue:** Vercel (CI/CD)

### Mi Rol en el Proyecto
Como responsable del proyecto, mi enfoque no fue solo la implementación, sino la **dirección técnica y el control de calidad**:
* **Diseño y Arquitectura:** Definición de la estructura de componentes y la jerarquía visual.
* **Auditoría de Código:** Revisión exhaustiva de cada módulo para asegurar el cumplimiento de las mejores prácticas de React y TypeScript.
* **Optimización de UI/UX:** Resolución de problemas complejos de responsividad y consistencia visual en dispositivos móviles.
* **Gestión de Flujo de Trabajo:** Implementación de un flujo de trabajo profesional mediante ramas (branching) y Pull Requests documentados.
